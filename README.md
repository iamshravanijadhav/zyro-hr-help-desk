# Zyro HR Help Desk

A Retrieval-Augmented Generation (RAG) chatbot built for the NIAT Masterclass RAG Challenge.

## Features

- Answers HR policy questions using company documents
- FAISS vector database
- HuggingFace embeddings
- Groq LLM (Llama 3.3 70B)
- LangSmith tracing and monitoring
- Retrieval-Augmented Generation (RAG)

## Documents Used

- Company Profile
- Employee Handbook
- Leave Policy
- Work From Home Policy
- Code of Conduct
- Performance Review Policy
- Compensation and Benefits Policy
- IT and Data Security Policy
- Prevention of Sexual Harassment Policy
- Onboarding and Separation Policy
- Travel and Expense Policy

## Tech Stack

- Python
- LangChain
- FAISS
- HuggingFace Embeddings
- Groq
- LangSmith

## Example Questions

- How many casual leaves are employees entitled to?
- What is the maternity leave policy?
- What is the notice period for L4 employees?
- Can earned leave be encashed?
